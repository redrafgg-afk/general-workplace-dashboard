# General Workplace Dashboard 🎓💼

**General Workplace Dashboard** is a lightweight, all-in-one productivity hub designed for students and professionals who need a fast, customizable workspace. Inspired by the VS Code aesthetic, it brings your essential academic and work tools into a single, efficient browser tab.

---

## ✨ Key Features

- 🔍 **Multi-Search Groups**: Group search engines by subject or project and query them all simultaneously.
- 🔗 **Labeled Interlinks**: Categorize research portals, e-learning sites, or work documents with clean, custom-labeled tags.
- 🧮 **Scientific Calculator**: Solve math, stats, and engineering problems instantly with built-in scientific functions.
- 🌐 **Quick Translator**: Instantly translate journal abstracts or technical terms using Google Translate.
- 💾 **Local Persistence**: Your data stays with you. Everything is saved in your browser's LocalStorage.

---

## ⚠️ Important: Browser Permissions

To use the **Multi-Search** feature (opening multiple search tabs at once), please **allow pop-ups** for this site in your browser settings.
1. Click the **lock/info icon** in the address bar.
2. Select **Site Settings**.
3. Set **Pop-ups and redirects** to **Allow**.

---

## 📥 Getting Started (Initial Configuration)

When you first open the online dashboard, it will be empty because the data is stored on *your* device, not a central server. To get my recommended setup:

1.  Navigate to the **`Save File`** folder in this repository.
2.  Download the **`config.json`** file.
3.  Open the Dashboard (via GitHub Pages or your local `index.html`).
4.  In the sidebar, click the **File Open (Import JSON)** icon.
5.  Select the `config.json` you just downloaded.
6.  **Done!** Your workspace is now pre-configured with my recommended academic setup.

## 🚀 How to Run

### 1. Online (GitHub Pages)
Simply visit: `https://your-username.github.io/general-workplace-dashboard/`

### 2. Local Desktop/Mobile
- **Desktop**: Download the files and open `index.html` in any browser.
- **Mobile**: Use the **Acode** app or any mobile web server to keep your data persistent.

---

## 🛠️ Customization
This dashboard is yours to modify. You can add, delete, or rename search groups and link categories directly through the UI. No coding knowledge is required. Remember to use the **Save (Export JSON)** button frequently to back up your custom configurations!
